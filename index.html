Начнем с Flow

Установка Flow

  1. Открой сайт https://flow.org/en/docs/install/
  2. Выбери пункт npm
  3. Выполни раздел Setup Flow по порядку (кроме npm install — это уже сделано)

Установка Flow

  1. Расставь флаг //@flowпервой строкой во всех js-файлах в директории src
  2. При запуске npm run flow не должно быть ошибок

Типы во Flow

  1. Примитивы: string, number, boolean
  2. any
  3. string[], { value:string }
  4. (id: string) => any;
  5. function foo(id: string): boolean {return true}

Enzyme

Пишем первый тест

Посмотри на компонент Question. Какие тесты к нему можно написать?

Пишем первый тест

			describe('<Question />', () => {
				it('должен генерировать две радио-кнопки', () => {
					const variants = [...]
					const wrapper = shallow(<Question
							variants={variants}/>);
					expect(wrapper.find('RadioButton').length)
				.to.equal(2);
				});
			});
