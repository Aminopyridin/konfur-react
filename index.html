Начнем с Flow

Установка Flow

  • Открываем сайт https://flow.org/en/docs/install/
  • Выбираем пункт npm
  • Выполняем все по порядку (кроме npm install — это уже сделано)
  • Расставляем флаг //@flowпервой строкой во всех js-файлах
  • При запуске npm run flow не должно быть ошибок

Storybook

Установка Storybook

  • Открываем сайт https://storybook.js.org/basics/slow-start-guide/
  • Выполняем все по порядку (кроме npm install — это уже сделано)
  • Сделаем все, чтобы сторибук запустился с дефолтным примером

Подготовимся к написанию собственной story

В конфиге сторибука надо сделать, чтобы он показывал все файлы, которые мы положим в папку stories. Сделаем это как-то так:

			const req = require.context('../stories', true, /\.js$/);
			req.keys().forEach((filename) => req(filename));

Enzyme

Пишем первый тест

Посмотри на компонент Question. Какие тесты к нему можно написать?

Пишем первый тест

Создай файл для теста в папке tests

			import React from 'react';
			import { expect } from 'chai';
			import { shallow } from 'enzyme';
			import sinon from 'sinon';
			import Question from '../src/components/Question';

Пишем первый тест

			describe('<Question />', () => {
				it('должен генерировать две радио-кнопки', () => {
					const variants = [{value: 'val1', label: 'text1'},
									  {value: 'val2', label: 'text2'}]
					const wrapper = shallow(<Question variants={}/>);
					expect(wrapper.find('RadioButton').length).to.equal(2);
				});
			});
